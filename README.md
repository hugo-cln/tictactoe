# tictactoe
Tic Tac Toe project in Javascript &amp; HTML
## Installation guide
### Prerequisites
- git for the installation (you can alternatively download the files from github)
- A compatible browser. Should work on any major & up-to-date browser with javascript enabled. Tested on Chrome 141 & Firefox 144.
---
1. Clone the repository
   - Open a command prompt in the desired install location
   - execute `git clone https://github.com/hugo-cln/tictactoe.git`
2. Open `index.html` in a supported browser with javascript enabled.
## How to play
1. You can select your pawn in the top drop-down (X is selected by default)
2. Click on "Jouer" below the grid.
3. You are always the first player to begin. Click on any empty cell to place your pawn.
4. Align 3 pawns in a row, column or diagonal to win.
5. Once the game is finished, you can change your pawn, and click on "Relancer la partie" to start a new game.

You can easily modify the size of the board by editing the `BOARD_SIZE` constant in `main.js`.
